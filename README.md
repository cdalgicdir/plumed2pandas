## Requirements

* Python 3.6+

## Installation

Install Plumed2Pandas with pip:

```sh
$ pip install Plumed2Pandas
```

or directly from the source code:

```sh
$ git clone https://github.com/cdalgicdir/plumed2pandas.git
$ cd plumed2pandas
$ python setup.py install
```
